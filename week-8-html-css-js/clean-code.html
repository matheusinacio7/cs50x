Clean Code

A Handbook of Agile Software Craftsmanship
by Robert C. Martin

A staple, this classic book covers very well the topic of writting understandable, readable and overall clean code (hence the name).

Clean Code is a widespread industry standard now, and this is the source.

In Portuguese, we have a saying "beber direto da fonte", which could translate both as "drink straight from the fountain" and as "drink straight from the source*" - which is a play with the meanings of the word "fonte" (either source or fountain).
